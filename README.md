# anvio_meta

Anvi'o is an open-source analysis and visualization platform for microbial 'omics data and this is a snakemake-based Anvi'o metagenomic workflow, heavily based on **sbx_anvio** and **sbx_coassembly** by Louis J Taylor https://github.com/louiejtaylor.

## Helpful refrences
- https://astrobiomike.github.io/metagenomics/metagen_anvio
- http://merenlab.org/2016/06/22/anvio-tutorial-v2/

## Required input files
- Fastq files
- A co-assembled contigs fasta file (from sbx_coassembly)

## Output
- Anvi'o interactive output
